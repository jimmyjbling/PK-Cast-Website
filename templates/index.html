ZINC Rx Property Calculator

This is an online web portal to estimate various drug properties described in PAPER LINK

Single SMILES will show results below. Similarity maps are generated with RDKit: Red means contributes to inactive class, green means contributes to active

Using similarity maps will increase the runtime significantly please be patient

When uploading a SMILES file make sure smiles are newline seperated (ei a csv with one column and each row is a SMILES). Uploaded files will auto-populate the text area, wait to generate csv until you see them in the box. Large numbers of SMILES and models wil take a long time to process, be patient. It is recommend that is number of SMILES * number of model is > 100 to instead use the standalone calculator LINK TO GITHUB instead of this webserver

Sorry, csv generation will not work on safari, use chrome, edge or firefox
